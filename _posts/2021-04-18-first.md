* 불필요 파일 삭제
  * https://mmistakes.github.io/minimal-mistakes/docs/quick-start-guide/#remove-the-unnecessary

  ```
  editorconfig
  gitattributes
  github
  /docs
  /test
  CHANGELOG.md
  minimal-mistakes-jekyll.gemspec
  README.md
  screenshot-layouts.png

  ```
